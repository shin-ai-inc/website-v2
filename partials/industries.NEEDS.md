# industries.html 必要画像メモ

## 結論
ラスタ画像は必須ではない。業種の視覚は線画のミニマル inline SVG と
淡いグラデ面（トークンの 3 ソリューション色）で成立させており、画像が無くても完成する。

## 任意（採用する場合のみ）
品位を一段上げたい場合に限り、次の補強が考えられる。なくても成立する設計を崩さないこと。

- `assets/images/industries-hero.webp`
  - 用途: page-hero 背景の淡い気配（`.page-hero__canvas` の代替/補助）。
  - 方向性: 抽象・上品。知識や網の目を想起させる静かな質感。人物・作り物ストック顔は不可。
    紙白〜淡い藍に寄せ、本文の可読性を損なわない低コントラスト。
  - ライセンス: 商用可・帰属不要（Unsplash / Pexels 等）。取得元は `assets/CREDITS.md` に記録。

## 注意
- 画像を追加する場合も BRIEF 4章の調達順と 9章の表記ルール（実績の断定回避）を守る。
- 現状の partial は画像参照を一切含まない。追加時は本メモを更新する。
