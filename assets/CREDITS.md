# 画像クレジット / ライセンス記録

このサイトで使用する画像の出所とライセンスを記録する。新規に画像を加える際は必ずここへ追記する。

## 自作・自社素材
- `assets/images/logo.png` — ShinAI ロゴ。現行公開サイト（shin-ai-inc/website）より継承。自社所有。
- `assets/images/ogp.png` — OGP画像（1200x630）。`_build/make_assets.py` で生成（藍のグラデ＋知識の網モチーフ）。自社制作。
- `assets/icons/favicon.svg` / `favicon.png` / `apple-touch-icon.png` / `icon-192.png` / `icon-512.png`
  — ブランドマーク（知識の網）。`_build/make_assets.py` および手書きSVGで生成。自社制作。

## フリー素材（使用時に追記）
現時点で外部のフリー画像は未使用。各ページは CSS グラデと線画 SVG で成立している。
今後フリー画像を加える場合は、次を必ず記録する。
- ファイル名 / 取得元URL / 作者 / ライセンス（商用利用可・帰属要否）/ 取得日。
- 採用基準: 革新性と信頼性が伝わる抽象的で上品なもの。作り物のストック顔写真やAIくさい定型画像は使わない。
